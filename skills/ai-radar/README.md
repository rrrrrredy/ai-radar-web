# Embedded AI Radar Skill

This folder contains a lightweight embedded copy of AI Radar Skill for project-local reference.

The canonical standalone version lives in the `ai-radar-skill` repository:

https://github.com/rrrrrredy/ai-radar-skill

Keep this embedded copy minimal. Update the standalone skill first, then mirror only the short project-specific guidance here.

Web Q&A and Skill Q&A are different:

- Skill Q&A uses the agent's tools and public evidence available during an agent session.
- Web Q&A will use Radar database retrieval first, then DeepSeek answer generation in a future phase.
